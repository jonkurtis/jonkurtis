I really enjoy the tech stack that I settled on for this site. It is built with [GatsbyJS](https://www.gatsbyjs.org)  hosted on [Netlify](https://www.netlify.com). The content is managed by [Netlify CMS](https://www.netlifycms.org) which allows me to write all of the content in markdown and host everything on my [GitHub Repo](https://github.com/jonkurtis/jonkurtis) without maintaining a server for the backend.
